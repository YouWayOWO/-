# 一、	計畫名稱：
圓筒形空氣淨化器本體
## 二、計畫起源：
一般市售空氣清淨機太過於昂貴性價比值偏低，過多錢投入來買品牌本身了，因此本小組專題立志打造高性價比之空氣淨化器。
## 三、計畫目標：
為製作性價比之空氣淨化器，我們的空氣淨化器只有簡單的本體，可讓消費者選擇自己想要使用的濾芯，此創新可讓消費者不必購買廠商的高利潤本體只需買廠商最核心的技術「濾芯」幫助消費者省錢。並擁有檢測當前環境空氣溫濕度與品質之功能。並整理成專案開源於Ｇithub上造福更多使用者的身體健康與錢包。
## 四、計畫理論：
購入「米家空氣淨化器濾芯」為濾芯範本，設計具有良好兼容性性上蓋，並在上蓋中加入120mm DC風扇並使用程式與按鈕進行控制風扇轉速調整，上蓋中整合顯示「空氣品質、溫度、濕度、風扇轉速」之功能，整套裝置使用12v電源供應器進行供電，以下為本空氣淨化器之線路圖。
## 裝置使用零件
| 零件名稱 | 功能 | 備註 |
| :--- | :--- | :---: |
| **米家空氣淨化器濾芯** | 示範濾芯 | 1 個 |
| **訂製上蓋 \* 2** | 主體 | 2 件 |
| **PMS5003** | 檢測 pm2.5 濃度 | 雷射感測器 |
| **120mm DC風扇** | 出風 | 動力來源 |
| **Arduino NANO** | 整合與控制各種硬體 | 主控板 |
| **LCD I2C** | 顯示狀態資訊 | 顯示介面 |
| **DHT11** | 檢測溫度與濕度 | 溫濕度感測 |
| **12V 2A電源供應器** | 為整套電路供應電力 | 外部電源 |
| **IRF520** | 提供與調節風扇電力 | MOS 驅動模組 |
| **DC電源轉接板** | 將電源進行轉接 | 電源分線 |
| **輕觸開關** | 用于調整風速 | 按鈕輸入 |
| **長螺絲與螺帽** | 本體之支撐 | 固定五金 |
